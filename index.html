Example Usages

Try typing any of the following expressions in a new cell and hit Enter:

    // Basic arithmetic and variable assignment
    2 + 2
    a = 5
    a * 10
    
    // Plotting functions
    plot('x^2', -10, 10)
    scatter('sin(x)', -6.28, 6.28)
    barPlot('cos(x)', 0, 6.28)
    histPlot([1, 2, 2, 3, 3, 3, 4, 4, 5])
    polarPlot('1+cos(theta)', 0, 6.28)
    
    // Algebraic operations and matrices
    
    // Solve a system of linear equations:
    //   2x + y = 1
    //   x  - y = 0
    A = [[2, 1], [1, -1]]
    b = [1, 0]
    lusolve(A, b)
    
    // Matrix operations
    m = [[1, 2], [3, 4]]
    m * m           // Matrix multiplication
    det(m)          // Determinant of m
    multiply(m, 10) // Scalar multiplication
    multiply(m, m)  // Element-wise multiplication
    
    // Advanced usage: simplify an algebraic expression
    simplify('x^2 - 4')
